💬 Estructura de Conversación con IA

Vamos a repasar un prompt y respuesta

1. System (Sistema)

Rol: Define el comportamiento del asistente
Ejemplo: "Eres un profesor experto en inteligencia artificial"

2. User (Usuario)

Rol: La pregunta o instrucción del usuario
Ejemplo: "Explícame qué es el fine-tuning"

3. Assistant (Asistente)

Rol: La respuesta del modelo
Ejemplo: "El fine-tuning es un proceso donde..."

📄 Formatos de Datos: JSON y JSONL

¿Qué es JSON?

  • JSON (JavaScript Object Notation) - Formato de datos estándar
  • Legible para humanos - Fácil de leer y escribir
  • Universal - Usado en casi todas las aplicaciones web
  • Estructura simple - Pares de clave-valor

📄 Ejemplo de JSON

{ "nombre": "María García", "edad": 30, "ciudad": "Madrid", "hobbies": ["leer", "cocinar", "viajar"] }

Elementos de JSON:

  • Llaves {} - Contenedor principal
  • Comillas "" - Para texto
  • Coma , - Separar elementos
  • Corchetes [] - Para listas
  • Dos puntos : - Separar clave de valor

📄 ¿Qué es JSONL?

JSONL (JSON Lines)

  • JSON por línea - Cada línea es un JSON completo
  • Formato para datasets - Ideal para grandes cantidades de datos
  • Fácil de procesar - Se lee línea por línea
  • Estándar en IA - Usado en fine-tuning
{"nombre": "María García", "teléfono": "123-456-789", "ciudad": "Madrid"}
{"nombre": "Juan López", "teléfono": "987-654-321", "ciudad": "Barcelona"}
{"nombre": "Ana Ruiz", "teléfono": "555-123-456", "ciudad": "Valencia"}

🔄 Diferencias: JSON vs JSONL

Característica JSON JSONL
Estructura Array completo Una línea = Un objeto
Tamaño Todo en memoria Procesamiento línea por línea
Fine-tuning No compatible ✅ Formato requerido
Edición Difícil para datasets grandes ✅ Fácil agregar/quitar

🛠️ Crear Archivos JSONL para Fine-Tuning

Formato para Chatbots (OpenAI):

Cada línea debe tener esta estructura:

{"messages": [{"role": "user", "content": "PREGUNTA"}, {"role": "assistant", "content": "RESPUESTA"}]}

Ejemplo práctico:

{"messages": [{"role": "user", "content": "¿Cuál es la garantía?"}, {"role": "assistant", "content": "2 años de garantía"}]}
{"messages": [{"role": "user", "content": "¿Cómo hago un reclamo?"}, {"role": "assistant", "content": "Envía un email a soporte@empresa.com"}]}

✅ Validación de Archivos JSONL

Reglas Importantes:

  1. Una línea = Un JSON completo - No dividir en múltiples líneas
  2. JSON válido - Cada línea debe ser JSON correcto
  3. Sin comas al final - Entre líneas no hay separadores
  4. Codificación UTF-8 - Para caracteres especiales
  5. Estructura consistente - Mismo formato en todas las líneas

Herramientas de Validación:

  • JSONLint: jsonlint.com - Validar JSON
  • Editor de texto - Verificar formato línea por línea
  • Python: Usar json.loads() para cada línea

🤖 Cómo Empezar con OpenAI

El camino más fácil para principiantes

Paso 1: Crear Cuenta

  1. Visita platform.openai.com
  2. Haz clic en "Sign up"
  3. Completa el registro con tu email
  4. Verifica tu cuenta por email

Paso 2: Configurar Facturación

  1. Ve a la sección "Billing"
  2. Agrega una tarjeta de crédito
  3. Establece un límite de gasto (recomendado: $5-10)
  4. Confirma la configuración

💡 Tip: Puedes empezar con $5 y ver cómo funciona

📋 Pasos 3 y 4 de OpenAI

Paso 3: Preparar tus Datos

  1. Crea un archivo JSONL con tus conversaciones
  2. Formato: {"messages": [{"role": "user", "content": "..."}, {"role": "assistant", "content": "..."}]}
  3. Mínimo 10 ejemplos (recomendado: 50-100)
  4. Datos de alta calidad y consistentes

Paso 4: Crear tu Fine-tuning

  1. Ve a la sección "Fine-tuning"
  2. Sube tu archivo JSONL
  3. Elige el modelo base (recomendado: gpt-3.5-turbo)
  4. Haz clic en "Create fine-tune"

⏱️ Tiempo: El entrenamiento puede tomar 10-30 minutos

🤗 Cómo Empezar con Hugging Face

La opción gratuita y flexible

Paso 1: Crear Cuenta

  1. Visita huggingface.co
  2. Haz clic en "Sign up"
  3. Completa el registro (gratuito)
  4. Verifica tu email

Paso 2: Crear tu Space

  1. Ve a "Spaces" en el menú
  2. Haz clic en "Create new Space"
  3. Elige un template de fine-tuning
  4. Configura el nombre y descripción

🆓 Ventaja: Completamente gratuito, solo necesitas una cuenta

📋 Pasos 3 y 4 de Hugging Face

Paso 3: Subir tus Datos

  1. Ve a la pestaña "Files" en tu Space
  2. Sube tu archivo de datos (CSV o JSONL)
  3. Formato: {"text": "pregunta", "label": "respuesta"}
  4. Confirma que se subió correctamente

Paso 4: Ejecutar el Entrenamiento

  1. Ve a la pestaña "App"
  2. Haz clic en "Run this Space"
  3. Monitorea el progreso en tiempo real
  4. Prueba tu modelo entrenado

⚡ Ventaja: Puedes ver el código y modificarlo si quieres
